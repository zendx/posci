## [1.0.0] - 2026-05-29

### Added
- Initial release of POSCI
- Complete Point of Sale (POS) system
- Inventory Management module
- Customer Management
- Supplier Management
- Sales Transaction processing
- Sales Returns management
- Purchase Returns management
- Accounts Receivable tracking
- Dashboard with analytics
- User authentication with Remember Me functionality
- Product categorization
- Financial transaction tracking
- Daily and monthly sales reports
- Bootstrap responsive UI
- jQuery-based interactive components
- DataTables for advanced table management
- CSV export functionality
- Print utility for transactions
- Admin panel with full CRUD operations

### Technical
- Built on CodeIgniter 3.x framework
- MySQL/MariaDB database support
- PHP 5.6.3+ compatible
- RESTful URL routing
- Custom libraries and helpers
- Session-based authentication
- Cookie-based Remember Me feature
